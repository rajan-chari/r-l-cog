This is the first post on **r-l-cog**. The layout is modeled after Hacker News:
a numbered, ranked list of posts with minimal chrome. Posts are written in
Markdown and built with Jekyll.

## Why this layout?

I wanted a blog that's:

- Fast to skim
- Text-first
- Easy to maintain — just drop a `.md` file in `_posts/`

## Writing a post

Create a file `_posts/YYYY-MM-DD-title.md` with frontmatter:

```yaml
---
layout: post
title: "Your title"
date: 2026-05-19
tags: [tag1, tag2]
---
```

Then write Markdown below.
